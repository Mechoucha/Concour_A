# Lexique administratif personnel

> **Objectif** : bâtir progressivement, en ~6 mois, un vocabulaire de cadre A territorial (gestion, gouvernance, politiques publiques, management, finances locales). Objectif cible **~200 entrées** d'ici les écrits.

## Règles du jeu

1. **Source principale** : *La Gazette des communes* ([lagazettedescommunes.com](https://www.lagazettedescommunes.com))
   — sources complémentaires : collectivites-locales.gouv.fr, CNFPT, rapports Cour des comptes.
2. **Rituel** : ~15 min/jour de lecture → à chaque lecture, noter **5 mots ou expressions** rencontrés qu'on ne maîtrisait pas.
3. **Format d'entrée** : mot → définition courte (de tes mots) → exemple de phrase concrète issue du texte.
4. **Révision** : les entrées entrent dans le système **Leitner** (J+1, J+3, J+7, J+21, J+60).
5. **Qualité > quantité** : mieux vaut 3 entrées bien comprises que 10 recopiées sans saisir.

## Modèle d'entrée

```
**Péréquation** — Mécanisme financier de redistribution des ressources entre collectivités pour réduire les inégalités de richesse fiscale.
*Exemple lu (Gazette, 18 avril 2026) : « La péréquation horizontale entre communes d'une même agglomération permet de compenser partiellement les écarts de potentiel fiscal. »*
```
