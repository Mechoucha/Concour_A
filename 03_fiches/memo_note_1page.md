## 1. Les 4 piliers (D-S-S-S)

- **D**estinataire → décideur (élu, DGS, chef de service)
- **S**ource → uniquement le dossier, rien hors dossier
- **S**ynthèse → reformulée, pas recopiée
- **S**tructure → plan apparent, titres et sous-titres

## 2. Les 3 étapes (à RENDRE VISIBLES sur la feuille)

1. **Mots-clés du sujet soulignés** (tous, sans en oublier)
2. **Tension formulée** avec la formule :
   *« Dans quelle mesure X **sans** Y ? »*
   → jamais *« Comment »*, jamais *« ou bien »*.
3. **Plan-roi écrit** :
   - **I = Constat** *(qu'est-ce qui se passe ?)*
   - **II = Perspectives** *(qu'est-ce qu'on peut faire ?)*

## 3. Le chapeau (3 temps obligatoires)

1. **Accroche ancrée terrain** (chiffre, fait, contexte FPT)
2. **Problématique** (la tension reformulée)
3. **Annonce de plan** : *« Après avoir... il conviendra de... »*

> Règle d'or : le chapeau **plante le décor et ferme le rideau**, il ne joue pas la pièce. 7 lignes max.

---

## 4. Les 4 pièges éliminatoires (à NE JAMAIS commettre)

| Piège | Symptôme | Antidote |
|---|---|---|
| **Plan miroir Cause/Solution** | I.A cause → II.A solution miroir, I.B cause → II.B solution miroir | I = phénomène (ampleur, formes), pas liste de causes |
| **Plan Avantages/Inconvénients ou Faits/Conséquences** | I = bien, II = mal | Constat / Perspectives, point. |
| **Glissement de verbe** | sujet dit *prévenir* → tu écris *remédier* | Re-souligner le verbe du sujet à l'étape 1 |
| **Hors-sujet partiel** | un mot-clé du sujet absent du plan | Étape 1 : tous les mots-clés cochés dans tes titres |

## 5. Tes 5 fautes-signatures (à bannir)

| Faute fréquente | Bonne orthographe |
|---|---|
| *térritoriales / térritorial* | **territoriales / territorial** |
| *colléctivités* | **collectivités** |
| *continuété* | **continuité** |
| *inigalités* | **inégalités** |
| *usagés* | **usagers** |

## 6. Relecture finale (non négociable)

- **À l'envers, mot à mot, crayon en main** (force le cerveau à voir chaque mot, pas le sens).
- Cible 1 : les 5 fautes-signatures ci-dessus.
- Cible 2 : accents, accords sujet/verbe, pluriels.
- Sanction : pas de copie envoyée sans relecture à l'envers.
